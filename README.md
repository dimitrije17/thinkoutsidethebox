####GAME DESCRIPTION###########

TYPE: PUZZLE
PURPOSE: 
USE ARROW KEYS TO MOVE THE CHARACTER
YOU HAVE TO FIND THE UNLOCK TO GET TO THE OTHER SIDE OF THE WALL

HINT: THINK OUTSIDE OF THE BOX! BOX BEING LITTLE GREAN MIDDLE BOX

*** Controls =======================================
Start  : Pause/Unpause
Select : Return to Start Screen
A      : Jump
B      : Interact
Left   : Move Left
Right  : Move Right
Up     : Move UP
Down   : Move Down


GameBoy | Keyboard
--------|----------

A       | Z
B       | X
L       | A
R       | S

Right   | Right Arrow
Up      | Up Arrow
Down    | Down Arrow
